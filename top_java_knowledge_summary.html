步骤1~8

一.没有仓库

1.初始化仓库: git init

2.拷贝一份远程仓库(下载一个项目):git clone https://github.com/zhi555/webTool.git

二.拥有仓库

3.下载远程代码并合并:git pull origin master

4.添加文件到暂存区(有新文件):git add .

5.查看仓库当前的状态:git status

6.提交暂存区到本地仓库:git commit -m "第一次版本提交"

7.下载远程代码并合并:git pull origin master

8.上传远程代码并合并:git push -u origin master
